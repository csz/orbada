4.2.2.2.1. Text box, simple

The field for entering data on single-line text.

ShortcutPopup menuDescription
Ctrl+Z (Alt+Backspace)Undo changesAllows you to undo changes
Ctrl+XCutCut the data from the field while putting them to the clipboard of the operating system
Ctrl+CCopyCopies selected data to the clipboard of the operating system
Ctrl+VPasteInsert the data at the cursor position from the operating system clipboard
Ctrl+ASelect allSelects all the data
HomeSets the cursor before the first char
EndSets the cursor to the last character in the box
