4.2.A.1. General

List the general keyboard shortcuts

ShortcutMenuDescription
Shift+F1Help | Available keyboard shortcutsList of all available at the moment, the keyboard shortcuts from menu
Ctrl+F2Perspective | Switch / restore full screenIf, on the screen is vertical or horizontal bar split screen, this function (available only with a shortcut) allows you to maximize or restore the portion on which the cursor is located.
Alt+0Perspective | Navigation | Select perspectiveIf you open more than one perspective, this function allows you to switch between them.
Alt+-Perspective | Navigation | Select a viewThis function lets you switch between views. This feature allows you to also open up new views from the available list.
Alt+=Perspective | Navigation | Select a tabThis function allows you to select a tab.

4.2.A.2. SQL Queries tab

A list of useful keyboard shortcuts available from the SQL queries tab

ShortcutMenuDescription
Ctrl+F8SQL Queries | SQL Command ListOpens a window from which to choose so far made a query and paste them into the editor.
Ctrl+GEdit | Incremental searchDisplays a dialog where you can find all occurrences of the string and move to the selected row.
Ctrl+Shift+AEdit | Select current textPoints selected, an active string, active SQL command.

4.2.A.3. For developers

A list of useful keyboard shortcuts for developers.

ShortcutMenuDescription
Ctrl+Shift+CEditor | Developer | Copy as program codeOpens a special window where you can parse a string as code.
