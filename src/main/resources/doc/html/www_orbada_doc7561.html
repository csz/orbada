4.2.2.1.1. A simple list of data

List of data not directly related to the database, which displays the data in static mode and program memory.
These functions are:

ShortcutPopup menuDescription
Ctrl+IAdjust the width of the dataAllows you to adjust the width of the columns to the data and column titles. The width is calculated on the basis of 50 rows below the highlighted line.
Shift+RightIncrease the widthncreases the width of the selected columns by 10 points on screen (pixels), not more than 1000
Shift+LeftReduce the widthReduces the width of selected columns by 10 points on screen (pixels), not less than 30
Shift+DownIncrease the row heightIncreases the height of the selected line of 10 points on screen, no more than 150
Shift+UpReduce the height of a rowReduces the height of the selected line of 10 points on screen, no less than the original amount thereof.
Alt+RightMove the column to the endAllows you to move the currently selected column at the end of the list.
Alt+LeftMove the last column in this placeAllows you to move the last column of the list in place of the currently selected.
Ctrl+CCopy value to clipboardAllows you to copy the value of the currently selected field to the clipboard of the operating system.
ExportA set of commands to export data from the list. These commands depend on the possession of plugins.
Ctrl+ESelect a columnWhen you select this option, a list of list columns from which you can choose the column on which you move the pointer.
Ctrl+EndJump at the end of the list or at the end of some lines read from the database.
Ctrl+Alt+EndJump to end of list data, if these figures are taken from the database, all rows are retrieved.
EnterDepending on the type of list, pressing Enter will move to the next column or next line.
Left, RightMove to the left, right columns list
Up, DownMove to the line above, below
Alt+AIf the list is associated with an action button, you can popup a menu of additional actions. These shares are associated with the available plugins in the program.

4.2.2.1.3. Fixed data list

Fixed list, you can view the data. This list has a fixed list of columns defined at the level of application design. Fixed lists represent, eg database tables and their properties. These lists are not opened on the user's request, for example, after the sql command.
Column layout, width and position is remembered.
Features that extend the functionality of the class lists of all lists is:

ShortcutPopup menuDescription
Ctrl+Shift+RRestore the original column propertiesrestore column layout from originally designed
Ctrl+VSearch by pasting from the clipboardAllows you to find the rows list on selected column, a string located in the system clipboard

4.2.2.1.4. Dynamic list of data

Dynamic list of data is dependent on the results made by the user (either directly or indirectly) a SQL command.
List of data stores only the width of columns and remamber it only on program runing.
The fonts and their families can be determined in the program settings.
