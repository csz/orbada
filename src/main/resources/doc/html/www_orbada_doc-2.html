Databases

ORBADA program, can connect to all databases, for which is a JDBC driver. Dedicated plug-ins let you manage the details of such databases as Oracle, Firebird, HSQLDB, DerbyDB, SQLite.

  • Oracle
  • Firebird
  • HSQLDB
  • Derby DB
  • SQLite
  • Microsoft SQL Servr
  • DBF
  • MySQL

Data exports

Export plug-ins allow to generate any kind of data to many external programs. Current tools allow you to generate data such as XML, HTML, SQL (INSERT), DBF, EXCEL, PDF, Text to the clipboard and much more.

  • To clipboard
  • As HTML file
  • As SQL Insert command
